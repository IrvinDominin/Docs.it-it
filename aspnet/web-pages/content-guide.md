<a name="webmatrix-content-guide"></a>Guida per il contenuto di WebMatrix
====================

> [!NOTE] 
> WebMatrix non è più consigliato come un ambiente di sviluppo integrato per ASP.NET Web Pages. Utilizzare [Visual Studio](xref:aspnet/web-pages/overview/getting-started/program-asp-net-web-pages-in-visual-studio) o [codice di Visual Studio](https://code.visualstudio.com/).

WebMatrix è un set di strumenti che fornisce un modo semplice per compilare siti Web di Microsoft Windows. Include IIS Express (un server web di sviluppo), ASP.NET (un framework di programmazione web) e SQL Server Compact (un database incorporato). Per visualizzare una breve panoramica di WebMatrix, vedere [Panoramica di WebMatrix](https://www.microsoft.com/web/webmatrix/). Per esaminare le funzionalità principali di WebMatrix, vedere [WebMatrix funzionalità](https://www.microsoft.com/web/webmatrix/features/).
